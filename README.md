## 기능 구현
- Form을 활용한 데이터 입력.
- Todo 체크, 삭제 기능
- (추가기능) Todo 별 색깔 지정.

## 학습 내용
- Form 컴포넌트 기초적인 기능.
- 배열 사용 방법 (push 대신 concat, map, filter).
- 리스트가 동적인 경우 함수형이 아닌 클래스형 컴포넌트로 작성해야 함. (컴포넌트 성능 최적화)
- react 개발자 도구 사용.
